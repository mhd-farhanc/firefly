## ğŸ Getting Started

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/mhd-farhanc/firefly.git](https://github.com/mhd-farhanc/firefly.git)
    ```
2.  **Install dependencies:**
    ```bash
    flutter pub get
    ```
3.  **Firebase Setup:**
    * This project relies on `firebase_options.dart`. You must configure your own Firebase project using:
    ```bash
    flutterfire configure
    ```
4.  **Run the app:**
    ```bash
    flutter run
    ```
